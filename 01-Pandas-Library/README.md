## 📌 Project Overview
This repository document my journey through the **Pandas** library, starting from fundamental data structures to complex feature extraction. As an AIML student, I focused on how these operations optimize data pre-processing—a critical step in the machine learning pipeline.

---

## 📂 Repository Structure
The project is organized into two primary sections to distinguish between conceptual modules and applied projects:

### 1. [01-Pandas-Foundations](./01-Pandas-Foundations)
This folder contains 7 detailed modules covering the essential "building blocks" of Pandas.
* **1) Series:** Understanding 1D labeled arrays and indexing logic.
* **2) DataFrames:** Mastering 2D data structures, slicing, and dicing.
* **3) Missing Data:** Industry-standard techniques for handling null values (`dropna`, `fillna`).
* **4) Merging & Joining:** Combining datasets using logic similar to SQL Joins.
* **5) GroupBy & Aggregation:** Summarizing data efficiently using split-apply-combine.
* **6) Pivot Tables:** Multi-dimensional data reshaping for reporting.
* **7) Operations:** Advanced functions like `apply()`, `unique()`, and sorting.

### 2. [02-Data-Analysis-Project](./02-Data-Analysis-Project)
This folder contains applied analysis and practical implementations on real-world datasets.
* **Feature Extraction:** A deep dive into transforming raw data into structured features for ML models.
